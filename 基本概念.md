# 註解

* html 註解方法:
```javascript
  
<!-- 在這裡填寫註解內容 -->

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
```

``` javascript
 // 註解內容 主要用於一個單字的註解

*註解內容 主要用於 一個大範圍的註解

```
